# Contributing to Quorum

Quorum is an early-stage research project. We are still reading papers 
and designing the architecture. This is intentional.

## What we need right now

**Research contributions**
- Papers or prior work relevant to TPM attestation, hardware trust, 
  hypervisor introspection, or behavioral biometrics
- Critiques of our assumptions — we want to be wrong early
- Knowledge of what existing tools already solve

**Technical contributions (coming soon)**
- Architecture will be published in docs/architecture/ once research 
  phase is complete
- Code contributions will open after architecture is finalized

## How to contribute now

Open an issue. Label it:
- research — pointing to papers or prior work
- question — challenging an assumption
- idea — suggesting an approach

## What we are not looking for yet

- Feature requests
- Code PRs (too early)
- Comparisons to existing tools without reading the problem statement first

## Principles

This project values depth over speed. We would rather take six months 
to understand the problem correctly than build the wrong thing quickly.
